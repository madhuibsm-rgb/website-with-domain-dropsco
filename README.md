# The Drops Co. — Coming Soon Site A single-page "coming soon" site. ## Files - `index.html` — the page - `logo.png` — your logo (add this — see step 1) ## 1. Add your logo Save your logo image into this folder and name it exactly **`logo.png`** (the page loads `logo.png` next to `index.html`). Upload it alongside `index.html` in the next step. ## 2. Put it on GitHub 1. Go to https://github.com/new and create a repo, e.g. `thedrops-coming-soon`. Keep it Public. Don't add a README (you have one). 2. On the new repo page, click **uploading an existing file**. 3. Drag in `index.html` and `logo.png` (and `README.md` if you like). 4. Click **Commit changes**. ## 3. Deploy on Vercel 1. Go to https://vercel.com and sign in with GitHub. 2. Click **Add New… → Project**. 3. Select your `thedrops-coming-soon` repo → **Import**. 4. Leave all settings as default (it's a static site, no build step needed) → **Deploy**. 5. After ~30 seconds you'll get a live URL like `thedrops-coming-soon.vercel.app`. To use a custom domain later: Vercel project → **Settings → Domains** → add your domain and follow the DNS steps. ## Updating later Edit the files on GitHub (or push new commits) — Vercel auto-redeploys on every change.
